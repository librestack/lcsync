# lcsync - Librecast Multicast Sync Tool

Work in progress.

Librecast file and data syncing tool.

Compare data with merkle trees, sync via multicast.

## Usage

Syncing local files:

`lcsync source destination`

Hexdump merkle tree:

`lcsync -hex file`

Outputs the hashes from base to root in hexadecimal.
